CSS Nesting

CSS Nesting allows you to write more readable and maintainable styles by nesting selectors within a parent element. This reduces redundancy and keeps related styles grouped together.


/* Without CSS Nesting */
.card {
    border: 1px solid #ccc;
    padding: 1rem;
    margin: 1rem;
}
.card .title {
    font-size: 1.5rem;
    color: #333;
}
.card .content {
    font-size: 1rem;
    color: #666;
}

            
/* With CSS Nesting */
.card {
    border: 1px solid #ccc;
    padding: 1rem;
    margin: 1rem;

    .title {
        font-size: 1.5rem;
        color: #333;
    }

    .content {
        font-size: 1rem;
        color: #666;
    }
}

CSS Grid

CSS Grid is a powerful layout system that allows you to create complex, responsive layouts with ease. It works by defining a grid container and placing items within the grid. For example: 


.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.item {
  background: #ccc;
  padding: 20px;
}

This code creates a grid with three equal-width columns and a 10px gap between items.

CSS Flexbox

CSS Flexbox is a one-dimensional layout model that allows you to align and distribute space among items in a container. It's particularly useful for creating flexible and responsive layouts. For example: 


.container {
  display: flex;
  justify-content: space-between;
}

.item {
  flex: 1;
  margin: 0 10px;
}

This code creates a flex container where items are spaced evenly and take up equal space.

CSS Variables (Custom Properties)

CSS Variables, also known as Custom Properties, allow you to store reusable values in your CSS. They can be defined globally or locally and are accessed using the var() function. For example: 


:root {
  --primary-color: #3498db;
}

.button {
  background-color: var(--primary-color);
}

This code defines a global variable --primary-color and uses it to style a button.

CSS Animations

CSS Animations allow you to create smooth transitions and animations on elements. You define keyframes to specify the styles at various points in the animation. For example: 


@keyframes slideIn {
  from {
      transform: translateX(-100%);
  }
  to {
      transform: translateX(0);
  }
}

.slide-in {
  animation: slideIn 1s ease-out;
}

This code creates an animation that slides an element in from the left.

CSS Transitions

CSS Transitions allow you to smoothly change property values over a specified duration. They are often used for hover effects or state changes. For example: 


.button {
  background-color: #3498db;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: #2980b9;
}

This code smoothly transitions the button's background color when hovered over.

CSS Pseudo-elements

CSS Pseudo-elements allow you to style specific parts of an element, such as the first letter or the content before or after an element. For example: 


p::first-letter {
  font-size: 2em;
  color: #e74c3c;
}

p::before {
  content: "Note: ";
  font-weight: bold;
}

This code styles the first letter of a paragraph and adds a bold "Note: " before the paragraph content.

CSS Pseudo-classes

CSS Pseudo-classes allow you to style elements based on their state or position in the document. Common pseudo-classes include :hover, :nth-child, and :focus. For example: 


a:hover {
  color: #e67e22;
}

li:nth-child(odd) {
  background-color: #f9f9f9;
}

This code changes the color of a link when hovered and applies a background color to every odd list item.
